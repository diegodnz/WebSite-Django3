Ativar virtual venv:
cd projeto
cd venv
cd Scripts
activate

. venv/bin/activate (linux e mac)

Criar o projeto Django:
django-admin startproject name

Rodar projeto Django:
python manage.py runserver

Parar servidor:
Ctrl + C

Criar tabelas no banco de dados:
python manage.py migrate

Criar superusuário:
python manage.py createsuperuser

Criação de app:
- Criar app: python manage.py startapp nameApp
- Registrar em installed apps
- Criar classe model -> Class nome(models.Model)
- Adicionar em admin.py -> admin.site.register(nomeModel)

Criar git repositório:
git init

Mostrar histórico do repositório
git status

Fazer um comit
git add .
git commit -m 'nome do commit'
- GitHub:
    git remote add origin URL
git push origin master (push -u -> primeira vez)
